# Overview

Artificial Intelligence (AI) refers to the simulation of human intelligence in machines which allows them to think and act like humans. It involves creating algorithms and systems that can perform tasks that require human abilities such as visual perception, speech recognition, decision-making and language translation.

## Basics

This section introduces the fundamental concepts of Artificial Intelligence and its core classifications.

### What is an AI Agent?

An AI agent is a software or hardware entity that performs actions autonomously with the goal of achieving specific objectives.

## Problem Solving

It is a fundamental aspect of AI which involves the design and application of algorithms to solve complex problems systematically.

### Search Algorithms

Search algorithms navigate through problem spaces to find solutions.

- Search algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Uniform Cost Search (UCS)
- Bidirectional search
- Greedy Best-First Search
- A Search* Algorithm

### Local Search Algorithms

It operates on a single current state and attempt to improve it incrementally by exploring neighboring states.

### Adversarial Search

It deal with competitive environments where multiple agents are in direct competition with one another.

### Constraint Satisfaction Problems

It is a problem-solving framework where variables have possible values and constraints restrict valid combinations of those values.

## Knowledge, Reasoning and Planning

It refers to the way information, knowledge and data are structured, stored and used by AI systems to reason, learn and make decisions.

### First Order Logic

It is used to represent knowledge and reason about the world by expressing objects, their properties and relationships in a structured way.

### Reasoning

It is the process by which AI systems draw conclusions, make decisions or infer new knowledge from existing information.

### Planning

It generates a sequence of actions an intelligent agent follows to achieve specific goals.

- Planning
- Forward State Space Search
- Markov Decision Processes (MDPs)
- Hierarchical State Space Search (HSSS)

**Uncertain Knowledge and Reasoning **

**Uncertain Knowledge and Reasoning**

It deals with handling incomplete or ambiguous information using methods designed to manage uncertainty.

- Uncertain Knowledge and Reasoning
- Dempster-Shafer Theory
- Probabilistic Reasoning
- Fuzzy Logic
- Neural Networks with dropout

**Types of Learning**

**Types of Learning**

Learning in AI refers to the process by which a system improves its performance on a task over time through experience, data or interaction with the environment.

### Supervised Learning

In this model are trained on labeled dataset to learn the mapping from inputs to outputs.

- Supervised Learning
- Linear Regression
- Logistic Regression
- Decision Trees
- Support Vector Machines (SVM)
- k-Nearest Neighbors
- Naïve Bayes
- Random Forests

### Semi-supervised learning

Here model uses both labeled and unlabeled data to improve learning accuracy.

**Unsupervised Learning**

**Unsupervised Learning**

Here the model is trained on unlabeled dataset to discover patterns or structures.

- Unsupervised Learning
- K-Means Clustering
- Principal Component Analysis (PCA)
- Hierarchical Clustering
- DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

### Reinforcement Learning

Here the agent learns through interactions with an environment using feedbacks.

- Reinforcement Learning
- Q-Learning
- Deep Q-Networks (DQN)
- Markov decision processes (MDPs)
- Bellman equation

### Deep Learning

It focuses on using neural networks with many layers to model and understand complex patterns and representations in large datasets.

- Deep Learning
- Neurons
- Single Layer Perceptron
- Multi-Layer Perceptron
- Artificial Neural Networks
- Feedforward Neural Networks
- Convolutional Neural Networks
- Recurrent Neural Networks
- Long Short-Term Memory networks
- Gated Recurrent Units Networks

### Probabilistic models

These models handles uncertainty by making predictions and supporting reasoning and decision-making from data.

- Probabilistic models
- Naive Bayes Classifier
- Monte Carlo Methods
- Expectation-Maximization (EM) Algorithm

## Generative AI

Generative AI focuses on creating new data examples that resemble real data, effectively learning the distribution of data to generate similar but distinct outputs.
